# Cortical Disinhibition

Experimenting with a model combining excitatory pyramidal cells and inhibitory parvalbumin (PV), somatostatin (SOM), and vasoactive intestinal peptide (VIP) cells.

it is a simplified version of the model by [Wagatsuma et al. (2023)](https://academic.oup.com/cercor/article/33/8/4459/6706754), using rate models rather than spikes.
